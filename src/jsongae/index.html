Talking JSON-RPC 2.0 with Google App Engine

Demo: GWT Client on GAE Python

This sample application demonstrates how to run a client written with GWT on Google App Engine (Python). The client communicates with the server interfaces through JSON/RPC (Version 2.0).

Remote Procedure Calls

In contrast to XML-RPC or SOAP, JSON-RPC allows for bidirectional communication between the service and the client, treating each more like peers and allowing peers to call one another or send notifications to one another. It also allows command batches to be sent to a peer.
